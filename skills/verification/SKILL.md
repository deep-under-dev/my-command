# 검증 루프 스킬

## 목적
Claude가 스스로 작업을 검증할 수 있게 함

## 원칙 (공식 문서 기반)

### 1. 검증 기준 먼저 정의
- 코딩 전에 성공 조건 명확히
- 테스트 케이스 예시 제공
- "이렇게 되면 성공" 정의

### 2. 자동 검증 활용
- 테스트 실행: `npm test`
- 린트: `npm run lint`
- 타입체크: `npm run typecheck`
- 빌드: `npm run build`

### 3. UI 검증
- 스크린샷 찍어서 비교
- 원본 디자인과 대조
- 차이점 리스트업

### 4. 근본 원인 해결
- 에러 억제 금지
- 왜 발생했는지 파악
- 재발 방지

## 안티패턴
❌ "잘 되는 것 같아요" (검증 없이)
❌ 에러 메시지만 숨기기
❌ 테스트 없이 "완료"
