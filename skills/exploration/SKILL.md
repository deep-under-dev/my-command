# 탐색 우선 스킬

## 목적
코딩 전에 먼저 이해하기

## 플로우 (공식 문서 권장)

### 1단계: 탐색
- 관련 파일 읽기
- 기존 패턴 파악
- 의존성 확인

### 2단계: 계획
- 구현 단계 설계
- 파일 변경 목록
- 테스트 계획

### 3단계: 확인
- 사용자에게 계획 공유
- 피드백 받기
- 필요시 수정

### 4단계: 구현
- 계획대로 진행
- 단계별 검증

## Plan Mode 활용
- 탐색/계획 시: Plan Mode 사용
- 구현 시: 일반 모드

## 질문 예시
코드베이스 이해할 때:
- "로깅이 어떻게 작동해?"
- "새 API 엔드포인트 어떻게 만들어?"
- "이 코드가 왜 foo() 대신 bar()를 호출해?"
