# /overview - 전체 프로덕트 파악

## 사용법
```
/overview [경로]
```

## 동작
1. 프로젝트 구조 분석
2. 주요 파일/디렉토리 파악
3. 기술 스택 확인
4. 아키텍처 요약

## 출력
```markdown
## 프로젝트 개요

### 기술 스택
- 언어: ...
- 프레임워크: ...
- 주요 라이브러리: ...

### 구조
```
src/
├── components/  # UI 컴포넌트
├── services/    # 비즈니스 로직
└── utils/       # 유틸리티
```

### 핵심 파일
- `src/index.ts` - 진입점
- `src/config.ts` - 설정

### 아키텍처
[간단한 설명]

### 빌드/실행
- 빌드: `npm run build`
- 테스트: `npm test`
- 실행: `npm start`
```

## 팁
- 처음 프로젝트 볼 때 먼저 실행
- 결과를 CLAUDE.md에 저장하면 좋음
