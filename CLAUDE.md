# CLAUDE.md

이 repo는 Claude Code 커맨드 & 스킬 모음입니다.

## 규칙

- 코딩 전 항상 계획 먼저 (/plan 또는 /build)
- 계획 후 /clear로 컨텍스트 정리
- 한 번에 한 단계씩 진행
- 각 단계마다 검증 (테스트/린트)

## 워크플로우

1. /overview - 프로젝트 파악
2. /build "기능" - 탐색 + 계획
3. /clear
4. /step N - 단계별 실행
5. /clear
6. 반복

## 토큰 절약

- 파일 전체 읽지 말고 필요한 부분만
- /clear 자주 사용
- 한 세션 = 한 작업
